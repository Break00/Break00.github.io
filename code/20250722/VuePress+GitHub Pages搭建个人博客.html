# VuePress+GitHub Pages搭建个人博客

# 一、项目准备

环境说明 Windows10+vscode

# 1.创建项目文件夹

如：demo文件夹（vscode打开文件目录）

# 2.全局安装VuePress

npm install -g vuepress

# 3.初始化项目

# 进入项目文件夹，在根目录下执行命令
npm init -y

# 4.更新配置文件

4.1 更新config.js文件

module.exports = {
  title: 'Hello',
  description: '我的个人网站',
  head: [ // 注入到当前页面的 HTML <head> 中的标签
    ['link', { rel: 'icon', href: '/logo.jpg' }], // 增加一个自定义的 favicon(网页标签的图标)
  ],
  base: '/', // 这是部署到github相关的配置
  markdown: {
    lineNumbers: false // 代码块显示行号
  },
  themeConfig: {
    nav:[ // 导航栏配置
      {text: '工作', link: '/work/' },
      {text: '生活', link: '/life/'}
    ],
    sidebar: 'auto', // 侧边栏配置
    sidebarDepth: 2, // 侧边栏显示2级
  }
};

4.2 更新package.json文件

"scripts": {
  "dev": "vuepress dev docs",
  "build": "vuepress build docs"
}

# 5.本地运行

npm run dev

可能出现的问题
Error: Cannot find module 'vue-template-compiler'
执行 npm install vue-template-compiler --save-dev

# 二、部署上线

说明：GitHub创建两个Repo，添加SSH Key

# 1.新建发布仓库

1.1 创建{USERNAME}.github.io仓库
USERNAME 必须是 Github 的账号名称 (公开权限)

# 2.新建代码仓库

仓库名无特殊要求（如：demo，可设置私有权限）
2.1 推送本地项目到代码仓库

  • 在项目根目录下新建deploy.sh文件
#!/usr/bin/env sh

# 确保脚本抛出遇到的错误
set -e

# 生成静态文件
npm run docs:build

# 进入生成的文件夹
cd docs/.vuepress/dist

# 如果是发布到自定义域名
# echo 'www.example.com' > CNAME

git init
git add -A
git commit -m 'deploy'

# 如果发布到 https://<USERNAME>.github.io (git初始化本地项目是master分支，github新建仓库默认是main分支)
git push -f git@github.com:Break00/Break00.github.io.git master:main

cd -
  • 推送项目（在项目根目录下执行）
git init
git add .
git commit -m "init"
git remote add origin git@github.com:Break00/demo.git

# git初始化本地项目是master分支，github新建仓库默认是main分支
git push -f git@github.com:Break00/demo.git master:main

2.2 更新package.json

"scripts": {
    "deploy": "bash deploy.sh"
}

2.3 部署上线

npm run deploy

在 Github Settings 中下面的链接: https://break00.github.io/ 即可看到博客主页
